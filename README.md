# 📱 Calculator App — SkillCraft Technology Internship  
### Task 1 — Android App Development

This is Task 1 of the SkillCraft Technology Android Development Internship, where I developed a fully functional Calculator App using Java and Android Studio. The app performs arithmetic operations and has AC, C, %, 00, and decimal features.

## 🚀 Task Title

## ✨ Features
- Basic arithmetic operations (+, -, *, /)
- Percentage (%) button
- Double zero (00)
- Clear last entry (C)
- Clear all (AC)
- Decimal support
- Modern UI design
- Real-time output display
